# Security Policy

## Supported Versions
Only the latest released version of the software is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v0.1.x  | :white_check_mark: |
| < v0.1  | :x:                |

## Reporting a Vulnerability

### Responsible Disclosure
If you find a security vulnerability, please report it privately. Do **not** disclose vulnerabilities publicly (such as by opening GitHub Issues) before a fix has been prepared and released.

### Contact Details
Please email security concerns to our team at:
`warishdeveloper@gmail.com`

### Expected Timeline
We will acknowledge your report within 48 hours and coordinate a timeline for releasing a patch.
